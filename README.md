# DDM-ModelLoader
A small library used to load in 3D models in c++

## Capabilities
### Model loading
Load in a single model from a file  
Supported file extensions:  
  - .obj  
  - .fbx  
  - .gltf  

### Scene loading
Load in a scene of models from a file  
Supported file extensions:  
  - .fbx  
  - .gltf
